# Proyecto CODER HOUSE
Hola! Este es mi proyecto para el curso de desarrollo web de Coder House. Este está armado con HTML, CSS, SCSS, Bootstrap.

La idea de esta pagina web es simplemente brindar información sobre lucha libre, más concretamente de la empresa WWE.
### Contenidos

- **Eventos**: Cartelera de todos los eventos PPV del año.
- **Estadisticas**: Mostrar de manera resumida los logros de algunos de los mejores luchadores de la empresa.
- **Campeones**: Los actuales campeones con su respectivo titulo(redirecciona a la sección estadisticas).
- **Horarios**: Informa acerca de los dias y horarios en los cuales se transmite el programa en los distintos paises de latinoamerica.
